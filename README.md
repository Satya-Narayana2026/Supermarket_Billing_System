# 🛒 Supermarket Billing System

A simple **Console-Based Supermarket Billing System** developed using **Python**. This application allows users to purchase grocery items, calculates the total bill along with GST, and generates a neatly formatted invoice.

## 📌 Features

- User-friendly command-line interface.
- Displays a list of available supermarket items with prices.
- Allows customers to purchase multiple items.
- Calculates the total amount based on item quantity.
- Automatically applies **5% GST** to the total bill.
- Generates a detailed invoice with:
  - Customer Name
  - Purchase Date & Time
  - Purchased Items
  - Quantity
  - Item Total
  - GST Amount
  - Final Payable Amount
- Handles unavailable items with appropriate messages.
