## Description
A console-based supermarket billing system developed in Python.

## Features
- Display product list
- Purchase multiple items
- Automatic bill generation
- GST calculation
- Date & time on invoice
- Multiple item support

## Technologies Used
- Python 3
- datetime module

## How to Run

python supermarket.py
